# sucrack

linux kullanıcılarının şifrelerini kırmak için kullanabileceğiniz bir araçtır kendileri

```
git clone https://github.com/willsh2/sucrack/
```

```
cd sucrack
```

```
python3 sucrack.py [username] [threads] [wordlist]
```
